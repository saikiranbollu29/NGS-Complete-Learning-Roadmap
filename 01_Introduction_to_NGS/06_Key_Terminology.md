# 🧬 Key Terminology in NGS and Genomics

## 📖 Introduction

Next-Generation Sequencing (NGS) and Genomics involve many technical terms that are used throughout sequencing experiments and bioinformatics analysis. Understanding these terms is essential for interpreting sequencing results, reading scientific literature, and answering interview questions.

This chapter provides a glossary of commonly used terms in genomics and NGS.

---

# 🧬 Basic Genetics Terms

| Term | Definition |
|------|------------|
| DNA | Deoxyribonucleic Acid, the hereditary material of most living organisms. |
| RNA | Ribonucleic Acid, involved in gene expression and protein synthesis. |
| Gene | A segment of DNA that contains instructions to produce a functional product (protein or RNA). |
| Genome | The complete set of genetic material in an organism. |
| Chromosome | A long DNA molecule packaged with proteins that carries many genes. |
| Nucleotide | The basic building block of DNA and RNA (A, T/U, G, C). |
| Allele | One of two or more versions of a gene. |
| Locus | The physical location of a gene or DNA sequence on a chromosome. |
| Mutation | A permanent change in the DNA sequence. |
| Variant | Any difference between a sample and the reference genome. |

---

# 🧬 NGS Terms

| Term | Definition |
|------|------------|
| Read | A DNA or RNA sequence generated by the sequencing machine. |
| Read Length | Number of bases present in one sequencing read. |
| Coverage (Depth) | Number of times a nucleotide is sequenced. |
| Reference Genome | A representative genome used for aligning sequencing reads. |
| Library | A collection of DNA or cDNA fragments prepared for sequencing. |
| Adapter | Short synthetic DNA sequence attached to DNA fragments during library preparation. |
| Barcode (Index) | A unique DNA sequence used to identify different samples in multiplex sequencing. |
| Multiplexing | Sequencing multiple samples together in a single run using unique barcodes. |
| Demultiplexing | Separating sequencing reads back into individual samples based on their barcodes. |
| Paired-End Sequencing | Sequencing both ends of the same DNA fragment. |
| Single-End Sequencing | Sequencing only one end of a DNA fragment. |

---

# 🧬 Bioinformatics Terms

| Term | Definition |
|------|------------|
| FASTQ | File format containing sequencing reads and quality scores. |
| FASTA | File format containing nucleotide or protein sequences. |
| SAM | Sequence Alignment/Map format; stores alignment information in text form. |
| BAM | Binary version of SAM, compressed and more efficient for analysis. |
| CRAM | Compressed alignment format that saves storage space. |
| VCF | Variant Call Format, used to store genetic variants. |
| GTF/GFF | Annotation file formats describing genes and genomic features. |
| BED | File format describing genomic intervals or regions. |

---

# 🧬 Variant Terms

| Term | Definition |
|------|------------|
| SNP (Single Nucleotide Polymorphism) | A change in a single nucleotide. |
| SNV (Single Nucleotide Variant) | A single nucleotide difference observed in a sample compared to the reference. |
| Indel | A small insertion or deletion in the DNA sequence. |
| CNV | Copy Number Variant; gain or loss of DNA segments. |
| SV | Structural Variant; large genomic rearrangement such as inversions or translocations. |
| Annotation | Adding biological information to identified variants. |

---

# 🧬 RNA-Seq Terms

| Term | Definition |
|------|------------|
| Transcriptome | The complete set of RNA molecules expressed in a cell or tissue. |
| Differential Expression | Comparison of gene expression levels between two or more conditions. |
| TPM | Transcripts Per Million; a normalized measure of gene expression. |
| FPKM | Fragments Per Kilobase of transcript per Million mapped reads. |
| RPKM | Reads Per Kilobase of transcript per Million mapped reads. |

---

# 🧬 Quality Control Terms

| Term | Definition |
|------|------------|
| Phred Score | A measure of base-calling accuracy. Higher scores indicate higher confidence. |
| Q20 | Approximately 99% base call accuracy. |
| Q30 | Approximately 99.9% base call accuracy. |
| GC Content | Percentage of guanine (G) and cytosine (C) bases in a sequence. |
| Adapter Contamination | Presence of sequencing adapter sequences within reads. |
| Duplicate Reads | Multiple identical reads originating from the same DNA fragment, often due to PCR amplification. |
